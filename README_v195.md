# Tribeca Aula v195 · Juego Izam 1.3 actualizado

Esta versión sustituye el juego del material 1.3 de Izam Sánchez por la versión mejorada `v21_5_random`.

## Archivos incluidos

- `service-worker.js`
- `assets/izam/tecnicas-estudio/unidad-1/gimnasio-1-pueblo-palabra/index.html`
- `assets/izam/tecnicas-estudio/unidad-1/gimnasio-1-pueblo-palabra/assets/`
- `supabase/sql/tribeca-aula-v195-izam-juego-material-1-3.sql`

## Instalación con GitHub Desktop

1. Descomprime este ZIP.
2. Copia el contenido descomprimido encima de la carpeta local del repositorio de Tribeca Aula.
3. Cuando Windows pregunte, acepta combinar carpetas y reemplazar archivos.
4. En GitHub Desktop, haz commit y push.

## Supabase

Si el material 1.3 ya estaba creado con la v194, no es obligatorio ejecutar SQL: la ruta del juego es la misma y basta con sustituir los archivos.

Si todavía no aparece como material 1.3, o quieres dejar el registro actualizado, ejecuta:

`supabase/sql/tribeca-aula-v195-izam-juego-material-1-3.sql`

## Comprobación

Abrir:

`TU_WEB/assets/izam/tecnicas-estudio/unidad-1/gimnasio-1-pueblo-palabra/index.html`

Si el juego carga directamente ahí, también debe funcionar dentro del aula de Izam como material 1.3.
